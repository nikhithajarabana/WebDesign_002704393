Assignment 1 A
The following HTML controls have been used in creating the 2 travel pages website.
1. A favicon has been added using the relationship attribute rel "shortcut icon" to get the distinguished icon on the browser.
2. A <table> tag has been used to create a table for the travel packages.
3. A Hyperlink has been given at the bottom of the home page which takes us to the emirates website for flight ticket booking.
4. A <form> has been used to create a registration form on the 2nd page of the website.
5. An <img> has been used to attach images on home page and second page
6. A <button> tag has been used in the form for submit.
7. External CSS style sheet has been used to style the HTML pages.
